****************
List Length Node
****************

.. figure:: /images/node-types_GeometryNodeListLength.webp
   :align: center
   :alt: List Length node.

The *List Length* node returns the number of items contained in a list.


Inputs
======

List
   The input list.


Properties
==========

Data Type
   The data type of the input list.


Outputs
=======

Length
   The total number of items in the input list.
