Set Active Group
================

.. reference::

   :Mode:      Edit Mode
   :Menu:      :menuselection:`Vertex --> Vertex Groups --> Set Active Group`

Sets the selected vertex group (from the menu) as the active group.

The active group is used by other operators that operate on the active vertex group.
