*******************
Integer Vector Node
*******************

.. figure:: /images/node-types_FunctionNodeInputIntVector.webp
   :align: right
   :alt: Integer Vector node.

The *Integer Vector* node outputs a constant integer vector value.


Inputs
======

This node has no inputs.


Properties
==========

Dimensions
   Number of components in the vector.

   Supports two or three-dimensional vectors.

Vector Values
   Integer values for each vector component.


Outputs
=======

Vector
   The generated integer vector value.
